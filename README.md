 
# RHCSA EX200 v10 Practice App

Hands-on training app for the **RHCSA (EX200 v10)** exam, based on realistic scenarios with an interactive terminal.

## Features

* 80 practical tasks
* Built-in terminal
* Automatic validation
* Hidden solutions (on demand or after failure)
* File editing tasks support
* Randomized task order
* Exam simulation (3h, score out of 300, passing score 210)

## Topics

* Core Linux commands
* Users and permissions
* System services
* Storage
* systemd timers
* Flatpak (RHEL 10)

## Usage

Read the scenario, type your commands, validate.

Some tasks include file editing with separate validation.
